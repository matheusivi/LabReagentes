# 🧪 LabReagentes

Aplicativo móvel desenvolvido com React Native (Expo) para controle de estoque de reagentes em laboratórios de análise clínica.

## 👥 Integrantes

- Matheus William de Oliveira

## 🎯 Objetivo

Facilitar o gerenciamento de reagentes laboratoriais, permitindo cadastrar, visualizar, editar e excluir reagentes com informações como validade, fabricante, lote, quantidade e localização. O app alerta visualmente sobre reagentes vencidos ou próximos do vencimento.

## 🏥 Contexto

Desenvolvido para a disciplina de Desenvolvimento Mobile, aplicado ao contexto de laboratórios de análise clínica, onde o controle de validade de reagentes é essencial para a segurança e qualidade dos exames realizados.

## 🛠️ Tecnologias Utilizadas

- React Native com Expo
- TypeScript
- expo-sqlite (banco de dados local)
- expo-camera (fotografia dos frascos)
- React Navigation (navegação entre telas)

## 📱 Funcionalidades

- Cadastro de reagentes com foto
- Alerta automático de validade (Ok / Vencendo / Vencido)
- Edição e exclusão de reagentes
- Persistência local com SQLite
- Lista ordenada por data de validade
